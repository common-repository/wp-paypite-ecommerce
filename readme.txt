== Installation ==

= Première méthode =

1. Cliquez sur "Plugins" dans le dashboard admin de votre site.

2. Cliquez sur "Ajouter nouveau dans les options des plugins.

3. Dans la barre de recherche, tapez "WP Payite Ecommerce".

4. Cliquez sur Installer puis Activez le plugin.

= Deuxième méthode =

1. Téléchargez le plugin à partir de <https://wordpress.org/plugins/wp-paypite-ecommerce>

2. Sauvegardez le plugin sur votre ordinateur et dézippez-le.

3. Connectez-vous au FTP ou Cpanel de votre site.

4. Téléversez le plugin dans le dossier wp-content/plugins folder.

5. Activez le plugin depuis le dashboard admin de votre site.

== Frequently Asked Questions ==

= Comment créer le bouton sur votre site ? =

C'est très simple, créez votre produit via la section "Ajouter nouveau", puis récupérez le shortcode dans la liste des produits et collez-le sur la page où vous voulez que la fiche produit doit s'afficher. 

= Et comment ça marche pour le client ? =

Le client doit également posséder un portefeuille Paypite et disposer des ressources nécessaires en paypites. Par la suite, il lui suffira de cliquer sur le bouton « Payer en paypites » et confirmer la transaction. Celle-ci sera enregistrée et affichée dans le portefeuille du e-commerçant et du client.

= C'est quoi l'URL de callback ? =

C’est l’URL à laquelle la personne devra être redirigée après le paiement. Si vous n'en spécifiez pas, sa valeur sera l'URL de la page sur laquelle le produit est affiché.

= Est-ce que vous proposez un support client? =

Oui, vous pouvez reporter les bugs, soucis ou propositions via la section support du plugin.
